## HC12T8 – Fusionner deux listes triées

```haskell
-- HC12T8

mergeLists :: Ord a => [a] -> [a] -> [a]
mergeLists xs [] = xs
mergeLists [] ys = ys
mergeLists (x:xs) (y:ys)
    | x <= y    = x : mergeLists xs (y:ys)
    | otherwise = y : mergeLists (x:xs) ys

main :: IO ()
main = print (mergeLists [1,3,5] [2,4,6])

```

## HC12T9 – Lire et afficher un fichier (avec gestion d’erreur)

```haskell
-- HC12T9

import System.IO
import Control.Exception

main :: IO ()
main = do
    putStrLn "Nom du fichier :"
    filename <- getLine
    content <- try (readFile filename) :: IO (Either IOException String)
    case content of
        Left _  -> putStrLn "Erreur : fichier introuvable."
        Right c -> putStrLn c

```

## HC12T10 – Module d’opérations mathématiques

MathOps.hs
```haskell
module MathOps (add, multiply) where

add :: Int -> Int -> Int
add x y = x + y

multiply :: Int -> Int -> Int
multiply x y = x * y

```

Main.hs
```haskell
-- HC12T10

import MathOps

main :: IO ()
main = do
    print (add 3 4)
    print (multiply 5 6)
```
